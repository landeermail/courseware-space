# 第23题：落管与上抛小球

## 题目概述

竖直放置的管（质量 2 kg，长 L = 24 m，上端 M、下端 N）受 16 N 竖直向上的拉力，从静止开始以 a = 2 m/s² 加速下落（取向下为正，g = 10 m/s²）。小球从管口 M 以初速度 v₀ 沿管轴竖直上抛。

- 第(1)问：管在足够高处，v₀ = 10 m/s，求小球从管底 N 穿出的时刻（答案 t = 4 s）。
- 第(2)问：管底 N 初始距地面 64 m，管落地（t′ = 8 s）时小球恰好落进管内，求 v₀ 范围（答案 29 ≤ v₀ ≤ 32 m/s）。

## 物理模型与公式

- 小球下落距离（自抛出点，向下为正）：h₁(t) = −v₀t + 5t²
- 管下落距离：h₂(t) = ½at² = t²
- 相对位移（小球相对管口 M，负值表示在 M 上方）：s(t) = h₁ − h₂ = −v₀t + 4t²
- 事件时刻：相对最高点 t = v₀/8；落回 M：t = v₀/4；穿出 N（s = 24）：t = (v₀ + √(v₀² + 384))/8
- 管参考系洞察：等效重力 g′ = g − a = 8 m/s²，相对运动即一次普通竖直上抛
- 第(2)问：t′ = √(2H/a) = 8 s，小球高度 y_b(8) = 8v₀ − 232，窗口 0 ≤ y_b ≤ 24 ⟺ 29 ≤ v₀ ≤ 32

## 交互说明

- **v₀ 滑块（5–40 m/s）**：核心交互。三维场景、y–t 图像、事件锚点与判定结论全部实时重算。
- **模式标签**：第(1)问（默认 v₀ = 10，无地面）/ 第(2)问（默认 v₀ = 29，含地面与 64 m / 88 m 初始高度标记）。
- **参考系切换**：地面参考系 ↔ 管参考系。管参考系中管保持静止，小球沿管运动（第(2)问地面会向上“冲”向管）。
- **y–t 图像**：小球 / 管口 M / 管底 N 三条高度曲线（管参考系下为 s–t 曲线与管内区间带），含事件交点与当前物理时间游标；第(2)问标出落地窗口 [0, 24] m 与“落进管内 / 落在管外”判定。
- **时间控制**：播放 / 暂停、可拖动时间轴、0.25×–2× 播放倍率；HUD 同时显示物理时间 t 与播放倍率。页面隐藏时自动暂停。
- **事件锚点**：一键跳转到抛出、相对最高点、落回 M、穿出 N（或管落地、小球落地）等关键时刻，时刻随 v₀ 重算。
- **三维场景**：半透明管身可直接看到管内小球，支持鼠标拖拽 / 触控旋转与缩放，提供正面 / 侧面 / 俯视视角预设。
