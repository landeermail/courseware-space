# 第23题：落管与上抛小球

## 题目概述

竖直放置的管（质量 2 kg，长 L = 24 m，上端 M、下端 N）受 16 N 竖直向上的拉力，从静止开始以 a = 2 m/s² 加速下落（取向下为正，g = 10 m/s²）。小球从管口 M 以初速度 v₀ 沿管轴竖直上抛。

- 第(1)问：管在足够高处，v₀ = 10 m/s，求小球从管底 N 穿出的时刻（答案 t = 4 s）。
- 第(2)问：管底 N 初始距地面 64 m，管落地（t′ = 8 s）时小球恰好落进管内，求 v₀ 范围（答案 29 ≤ v₀ ≤ 32 m/s）。

## 物理模型与公式

- 小球下落距离（自抛出点，向下为正）：h₁(t) = −v₀t + 5t²
- 管下落距离：h₂(t) = ½at² = t²
- 相对位移（小球相对管口 M，负值表示在 M 上方）：s(t) = h₁ − h₂ = −v₀t + 4t²
- 事件时刻：相对最高点 t = v₀/8；落回 M：t = v₀/4；穿出 N（s = 24）：t = (v₀ + √(v₀² + 384))/8
- 管参考系洞察：等效重力 g′ = g − a = 8 m/s²，相对运动即一次普通竖直上抛
- 第(2)问：t′ = √(2H/a) = 8 s，小球高度 y_b(8) = 8v₀ − 232，窗口 0 ≤ y_b ≤ 24 ⟺ 29 ≤ v₀ ≤ 32

## 交互说明

单屏工作台布局：左侧三维示意场景，右侧 y–t（或 s–t）图像，底部一行控件。桌面与 iPad 横屏一屏看完；手机竖屏自动上下堆叠。

- **叙事结构**：默认第(1)问故事模式（管参考系，v₀ = 10），播放以 0.5× 演示倍率运行，并在事件点自动暂停——相对最高点 → 落回 M → 穿出 N；第(2)问为探究模式（地面参考系，v₀ = 29），v₀ 滑块升格为主控件并显示窗口判定。
- **v₀ 滑块（5–40 m/s）**：核心交互。三维场景、图像、事件刻度与判定结论全部实时重算。
- **参考系切换**：地面参考系 ↔ 管参考系。管参考系中管保持静止，小球沿管运动（第(2)问地面会向上"冲"向管）。
- **播放 + 时间轴合一**：事件以彩色圆点直接标在时间轴上，点击跳转；HUD 同时显示物理时间 t 与演示倍率 0.5×。页面隐藏时自动暂停。
- **y–t / s–t 图像**：地面系为小球 / 管口 M / 管底 N 三条高度曲线；管系为 s–t 曲线与管内区间带 [0, 24] m；第(2)问标出落地窗口与"落进管内 / 落在管外"判定。图像下方方程行始终给出地面系真实数值。
- **三维场景为示意图**：纵向压缩 4×（管显示长为实际 1/4），相对位置关系保持精确，数值以图像和标注为准；半透明管身可直接看到管内小球，默认侧面正视角，支持拖拽旋转与缩放。
