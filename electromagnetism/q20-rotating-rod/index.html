先认清谁在运动

圆环内的金属棒 OP 绕 O 点顺时针转动；导体棒 CD 横跨两根倾斜导轨。先拖动画面，找到 O、P、M₁、M₂ 和 CD。
OP 运动→产生电动势→回路电流→CD 受力

从上方看 OP 顺时针转动、磁场竖直向上。你预测哪一点电势更高？

O → M₁ → CD → M₂ → 圆环 → P

外电路的传统电流由高电势 O 流回 P；在 CD 中从 M₁ 侧流向 M₂ 侧。

CD 沿斜面方向同时受到重力分力、摩擦力和安培力。电流方向与磁场共同决定安培力沿导轨向上。

静摩擦力会在需要的方向上调节，因此“能静止”对应一个安培力范围，而不是单一数值。
