⚡ 核心原理

OP 旋转切割磁感线：金属棒 OP 在竖直向上的匀强磁场中绕 O 点转动，产生感应电动势

E = ½ Bωr²

回路电流：OP 与 CD 串联，总电阻 2R，电流 I = E/(2R)

CD 受力：安培力 F安 = BIL，方向沿导轨向上（由电流方向决定）

🔒 第(1)问：静力平衡

CD 静止在磁场中，需满足：

mg sin37° − μmg cos37° ≤ F安 ≤ mg sin37° + μmg cos37°

代入数据：0.4N ≤ F安 ≤ 2.0N

对应角速度：8 rad/s ≤ ω ≤ 40 rad/s

💡 提示：当 ω 在 8~40 rad/s 范围内时，安培力恰能平衡重力分量和摩擦力的组合，CD 保持静止。

🚀 第(2)问：匀加速运动

CD 以 v₁=2m/s 进入磁场，做匀加速运动 a=1m/s²：

① ω(t) = (16 + 10t) rad/s
② 外力对 OP 做功 W = −0.42 J

OP 角速度随时间线性增加，以维持回路中恒定电流 I₁=2A。
